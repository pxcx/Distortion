# Distortion
VST3 Plugin simulating a simple distortion guitar pedal, built with JUCE framework.

Based on [**Reon Fourie's Tutorial**](https://www.youtube.com/watch?v=iNCR5flSuDs&ab_channel=ReonFourie)

![Plugin UI](https://pxcx.com.br/github/distortion.png)

### Requirements

To build this project you will need
 - [**JUCE framework**](https://juce.com/get-juce/download).
 - [**Steinberg's ASIO SDK**](https://www.steinberg.net/en/company/developers.html). 

### Paths

This project will search for JUCE modules and ASIO SDK in the following paths:

- JUCE modules: C:\Program Files\JUCE\modules
- ASIO SDK: C:\SDKs\asiosdk_2.3.3_2019-06-14
